# Control Bank

Realiza o controle do sistema de banco

## Implementação

| Endpoint                    | Método | Descrição                                           |
|-----------------------------|--------|-----------------------------------------------------|
| `/clientes/[id]/extrato`    |  GET   | Pega o extrato de transações do cliente             |
| `/clientes/[id]/transacoes` |  POST  | Realiza a transação para determinado cliente        |
